# 页面开发维护说明
> views 下创建项目公共内容组件页面
> 大量通用页面，按组件开发，并书写使用说明；（如 main ）
> 关于 main ，页面格式为 当前位置 + 筛选条件 + 列表；（使用方法查看 main 开发维护）

## 目录
```
- views
  ├ login 登录 
  ├ layout 布局
  ├ errorPage 错误页面
  ├ dashboard 首页
  ├ main 内容页（常见格式）
  ├ pages 功能页面
    └ ... 
```

### main 开发维护
> 页面内容 当前位置 + 筛选条件 + 列表，三个模块，各模块各自维护；
> 当前位置 + 筛选条件 是否可用，默认true;
> 使用页面，需传入 方法 + 内容 等，详细查看 main 文件夹下 README.md
